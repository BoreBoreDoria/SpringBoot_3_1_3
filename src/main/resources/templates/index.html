    Информация о приложении

    Приложение разработна в рамках выполнения лабораторной работы по дисциплине "Информационный менеджмент

    Использован язык программирования Java и JavaScript для обновления эллементов

    Выполнил работу Рочев Н.А., студент группы З-99э-1




    Приложение необходимо для инвентарного учёта серверов

    Были созданы следующие функции:

    1) авторизация на 2 роли. Пользователь (доступен только просмотр) и администратор (может добалвять и изменять данные)

    2) создание, удаление, редактировние пользователя

    3) создание, удаление, редактировние сервера

    4) Хранение данных в базе
